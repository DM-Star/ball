# 弹球游戏

这是一款基于WIN32 API实现的弹球游戏，支持冒险模式、重力模式、试炼模式、生存模式。

## QuickStart

**在Windows系统下：**

运行`Test/ball.exe` 可直接运行示例

**编译方法：**

将`OpenGL-Final.sln`导入VS2015打开项目并编译
